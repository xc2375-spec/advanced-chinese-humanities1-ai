Focus on Patterns

Learn through radicals and character clues

1. 扌 often points to hand action

In this lesson, several characters with 扌 involve direct hand action: adding, grabbing, lifting, or opening.

添tiānadd
抢qiǎnggrab
举jǔraise
掀xiānlift / open
Memory tip: when you see 扌, ask whether the word involves a hand action, physical manipulation, or direct control.
In this lesson, what is the best reason to notice 扌 in 添, 抢, 举, and 掀?

2. Clothing and body words can form a memory cluster

The second story contains a strong appearance cluster. Learning these words together makes them easier to remember.

帽子màozihat
裙子qúnziskirt
头巾tóujīnheadscarf
鼻子bízinose
Memory tip: build a “body + clothing” network. If you remember one item, it helps you retrieve the others in the same semantic group.
Why is it helpful to study 帽子, 裙子, 头巾, and 鼻子 as a cluster?

3. Pair the result words: 得意 → 后悔

Some lesson words are easier to remember as a sequence of events or emotional change.

得意déyìpleased with oneself
后悔hòuhuǐregret
多余duōyúunnecessary
反而fǎn'érinstead
Memory tip: in idiom stories, emotional sequence matters. Here the logic is: overconfidence → unnecessary action → opposite result → regret.
What is the most useful learning pattern in 得意, 多余, 反而, and 后悔?

4. Observe contrast: 仔细 vs 马马虎虎

Contrast pairs help build a sharper vocabulary network.

仔细zǐxìcareful
马马虎虎mǎma-hūhūso-so / careless
观察guāncháobserve
犹豫yóuyùhesitate
Memory tip: pairing a precise word with a loose or casual one helps you remember both more clearly.
Why is 仔细 useful to compare with 马马虎虎?
